# Linuxscreen

> Linuxscreen is **not** currently legal for use in [speedrun.com](https://www.speedrun.com/mc) submissions or [MCSR Ranked](https://mcsrranked.com/) matches.

Linuxscreen is a Minecraft Java Edition screen mirroring and overlay tool for heavily inspired by [Toolscreen](https://github.com/jojoe77777/Toolscreen), and an alternative of it for Linux and macOS. It is meant to bring things like mirrors, overlays, rebinds, sensitivity changes, EyeZoom, and more directly into the game without relying on OBS projectors or other external tools.
